Important Advisory | Khuyến cáo quan trọng
1. Target Audience | Đối tượng nên và không nên sử dụng
EN: Only for Low-Spec or Older PCs. Designed for office computers or budget student laptops with small RAM capacity (<6GB) and weak processors . Mid-to-high-spec computers do not need this tool since Windows already manages their resources efficiently.
VN: Chỉ dùng cho máy cấu hình thấp hoặc đời cũ. Được thiết kế chuyên biệt cho máy tính văn phòng hoặc laptop học tập giá rẻ có dung lượng RAM nhỏ (<6GB) và bộ vi xử lý yếu . Máy tính cấu hình trung bình trở lên không cần sử dụng vì hệ điều hành Windows đã tự quản lý rất tốt.
2. Strictly Not for Gaming | Tuyệt đối không phục vụ để chơi game
EN: This is not a game booster. Gaming requires hardware to run at maximum capacity continuously. This tool's resource-shifting mechanism will not increase your FPS, reduce game lag, or turn a weak computer into a gaming rig.
VN: Đây không phải là phần mềm tăng tốc game. Chơi game đòi hỏi phần cứng hoạt động liên tục với công suất tối đa. Cơ chế luân chuyển và đóng băng tài nguyên của công cụ này không giúp tăng FPS hay giảm lag khi chơi game, thậm chí có thể gây xung đột ngoài ý muốn.
3. Sole Objective | Mục tiêu duy nhất
EN: To ensure smooth performance for basic daily tasks like office work (Word, Excel), web browsing, light entertainment (videos, movies), and communication (Zalo, Messenger, Discord, Viber).
VN: Giúp các máy tính và laptop cũ xử lý mượt mà nhất các tác vụ văn phòng cơ bản (Word, Excel), lướt web học tập, xem phim giải trí nhẹ nhàng và nhắn tin trao đổi công việc hàng ngày.
Introduction | Giới thiệu
EN: When multiple programs run at once on low-spec computers, Windows tries to distribute power equally, making your active application laggy. Neko Optimizer acts as a "smart butler" to solve this by focusing resources where they are needed most.
VN: Khi nhiều phần mềm chạy cùng lúc trên máy yếu, Windows cố gắng chia đều sức mạnh khiến ứng dụng bạn đang dùng trực tiếp bị giật lag. Neko Optimizer đóng vai trò như một "quản gia thông minh" giải quyết vấn đề này bằng cách dồn tài nguyên cho ứng dụng bạn đang tương tác.

2. Smart Exception List (Whitelist) | Danh sách ngoại lệ thông minh
Web Browsers (Chrome, Coc Coc):
EN: Only reduce resources when idle; never freeze to avoid page crashes.
VN: Chỉ giảm tài nguyên khi nhàn rỗi; không bao giờ đóng băng để tránh lỗi tải lại trang hoặc sập trình duyệt.
Music Apps (Spotify, YouTube):
EN: Never frozen or memory-reclaimed to ensure uninterrupted music.
VN: Không bao giờ đóng băng hay dọn dẹp RAM để nhạc luôn phát liên tục không ngắt quãng.
Messaging Apps (Zalo, Viber, Discord):
EN: Allowed to hibernate but automatically "wakes up" every 30 seconds to receive messages.
VN: Cho phép ngủ đông nhưng cứ mỗi 30 giây sẽ tự động thức dậy để kiểm tra và nhận tin nhắn mới rồi lại ngủ tiếp.
3. Absolute System Safety | Bảo vệ hệ thống an toàn
EN: Does not touch Windows core processes to prevent blue screens. Smart Chrome filter protects GPU/render components from crashing.
VN: Không can thiệp vào các tiến trình cốt lõi của Windows để tránh lỗi màn hình xanh. Bộ lọc thông minh bảo vệ Chrome không bị lỗi đen màn hình hay sập trình duyệt.
Quick Start Guide | Hướng dẫn sử dụng nhanh
1. How to Run | Cách mở lần đầu tiên
EN: Keep optimizer.exe in the same directory as whitelist.txt.                                                                                                                         VN: Để file optimizer.exe cùng thư mục với file whitelist.txt.
EN: Right-click optimizer.exe -> Select Run as administrator (Required once to register auto-start task).                                             VN: Nhấp chuột phải vào optimizer.exe -> Chọn Run as administrator (Chỉ cần làm một lần đầu để tự động đăng ký chạy cùng Windows).
2. Console Hotkeys | Các phím tắt nhanh
SPACE: * EN: Pause or resume the optimization process.
VN: Tạm dừng hoặc tiếp tục quá trình tối ưu hóa.
H: * EN: Hide this window to the system tray.
VN: Ẩn cửa sổ này đi để chạy ngầm dưới khay hệ thống (System Tray).
R: * EN: Reload the exception list whitelist.txt without restarting.
VN: Đọc lại file ngoại lệ whitelist.txt mà không cần bật lại phần mềm.
ESC: * EN: Restore all hibernating software to active states and exit.
VN: Trả mọi phần mềm đang ngủ đông về trạng thái bình thường và thoát hoàn toàn.
